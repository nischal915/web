Web Technology I

Web development is the process of building websites and web applications. It involves a combination of front-end and back-end technologies to create interactive and dynamic content.

1. Introduction to Web Development

Web development refers to the tasks associated with the development of websites. It includes everything from designing web pages to building applications that run on the internet.

1.1 Web Browsers and Search Engines

Web browsers allow users to access web pages. Some of the most popular web browsers include Google Chrome, Mozilla Firefox, and Safari. Search engines like Google and Bing help users find websites and content on the web.

1.2 Overview of Various Internet & Web Technologies

  • HTML: The standard markup language for creating web pages.
  • CSS: A language used to style HTML documents and make them look appealing.
  • JavaScript: A programming language that enables interactive effects and dynamic content on websites.
  • PHP: A server-side scripting language used for creating dynamic web pages.
  • Database Technologies: MySQL, PostgreSQL, and MongoDB are some of the databases used to store data for websites.

2. Content Management System (CMS)

A Content Management System (CMS) is a software application that allows users to create, manage, and modify content on a website without requiring specialized technical knowledge.

2.1 CMS Examples

  • WordPress: One of the most popular open-source CMS platforms used for blogging and website creation.
  • Joomla: Another open-source CMS known for its flexibility and scalability.
  • Drupal: A highly flexible CMS used for more complex websites and applications.

3. HTML: The Language of the Web

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure of web pages, allowing browsers to render content properly.

3.1 Structure of HTML

HTML documents consist of elements that define the structure of the page. These elements are enclosed within angle brackets (e.g., <div>, <body>).

  • HTML Tags: Tags are the building blocks of HTML. Examples include <html>, <head>, and <body>.
  • Attributes: HTML tags can have attributes that modify their behavior. For example, <a href="URL"> defines a link to a specific URL.

3.2 Published and Hosting

Publishing a website means making it accessible on the internet. This is done through web hosting services. Examples of web hosting services include Bluehost, GoDaddy, and AWS.

3.3 HTML Tags vs. Attributes

HTML tags define the elements of the web page, while attributes provide additional information about the elements. For example, the <img> tag displays an image, and the src attribute specifies the image file.

3.4 Basic Tags of HTML

  • <html>: Defines the root of the HTML document.
  • <head>: Contains metadata about the document, such as the title and links to external files.
  • <body>: Contains the content of the web page that is visible to the user.
  • <title>: Specifies the title of the web page, shown in the browser tab.

3.5 Heading Tags

Heading tags are used to create headings on a web page, with <h1> being the most important and <h6> the least important. The syntax for heading tags is as follows:

<h1>Main Heading</h1>

3.6 Paragraph Formatting

The <p> tag is used to define a paragraph in HTML. For example:

<p>This is a paragraph.</p>

3.7 Break Line

The <br> tag is used to insert a line break within text, allowing content to be displayed on the next line.

3.8 Comment in HTML

Comments in HTML are written between <!-- and -->. They are used to annotate the code without affecting the display on the page.

<!-- This is a comment -->

3.9 Formatting Text

HTML provides several tags for formatting text:

  • <b>: Bold text.
  • <i>: Italic text.
  • <u>: Underlined text.
  • <mark>: Highlighted text.
  • <sup>: Superscript text.
  • <sub>: Subscript text.
  • <em>: Emphasized text.

3.10 Lists in HTML

  • Ordered List (<ol>): Displays a numbered list. Example: 
    <ol>
                    <li>Item 1</li>
                    <li>Item 2</li>
                </ol>
  • Unordered List (<ul>): Displays a bulleted list. Example: 
    <ul>
                    <li>Bullet 1</li>
                    <li>Bullet 2</li>
                </ul>

4. Creating Links in HTML

Links in HTML are created using the <a> tag. Links can point to other pages within the same website or external URLs:

  • Link to another page: 
    <a href="page2.html">Go to Page 2</a>
  • Link to an external website: 
    <a href="http://example.com">Visit Example</a>

5. HTML Forms

HTML forms are used to collect user input. They include various form elements such as text boxes, radio buttons, checkboxes, and buttons:

  • Textbox: <input type="text">
  • Radio Buttons: <input type="radio">
  • Checkboxes: <input type="checkbox">
  • Buttons: <button>

5.1 Example of a Basic Form

<form action="submit.php">
            <input type="text" name="username">
            <input type="submit" value="Submit">
        </form>

6. Cascading Style Sheets (CSS)

CSS is used to style HTML elements and control the layout of a web page. It separates the content (HTML) from the style and design (CSS).

6.1 Introduction to CSS

CSS (Cascading Style Sheets) is used to define the look and feel of a web page. With CSS, you can change the color, font, size, spacing, and position of elements on a page.

6.2 Inline CSS

Inline CSS is used to apply styles directly within an HTML element using the style attribute. Example:

<p style="color: red;">This text is red</p>

6.3 Embedded CSS

Embedded CSS is written within the <style> tag in the <head> section of the HTML document. Example:

<style>
            p { color: blue; }
        </style>

6.4 External CSS

External CSS is written in a separate file with a .css extension and linked to the HTML document. Example:

<link rel="stylesheet" href="styles.css">
