Web Technology-I (HTML + CSS)

HTML (Hypertext Markup Language)

HTML is used to structure content on the web. It uses elements like:

  • <html>: The root element of an HTML document.
  • <head>: Contains meta-information like the title and link to stylesheets.
  • <body>: Contains the visible content of the webpage.

CSS (Cascading Style Sheets)

CSS is used to style HTML content, such as setting colors, fonts, and layouts. CSS selectors target HTML elements to apply styles.
