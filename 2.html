Number System and Conversion (Boolean Logic)

What is a Number System?

The number system is a way of representing numbers using digits or symbols. In computing, common number systems include:

  • Binary: Base-2 system, consisting of two symbols: 0 and 1.
  • Decimal: Base-10 system, the standard system for denoting integer and non-integer numbers.
  • Hexadecimal: Base-16 system, often used in programming to represent large binary numbers more compactly.

Boolean Logic

Boolean logic is fundamental in computing for decision-making and circuit design. Key Boolean operations include:

  • AND: Results in true if both operands are true.
  • OR: Results in true if at least one operand is true.
  • NOT: Inverts the value of the operand.
  • XOR: Results in true if only one operand is true, but not both.
