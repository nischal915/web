Unit 2: Number System and Boolean Logic

Introduction to Number Systems

Computers use different number systems to represent data. These systems include decimal, binary, octal, and hexadecimal.

Decimal Number System

The decimal system is base 10, used in everyday arithmetic. It includes digits from 0 to 9.

Binary Number System

The binary system is base 2, used by computers for processing. It includes only two digits: 0 and 1.

Octal Number System

The octal system is base 8, sometimes used in programming. It includes digits from 0 to 7.

Hexadecimal Number System

The hexadecimal system is base 16, commonly used in programming and web design. It includes digits from 0 to 9 and letters A to F.

Conversion Between Number Systems

Converting between these systems is essential for programming and computer operations:

  • Decimal to Binary: Divide by 2 until quotient is 0.
  • Binary to Decimal: Multiply each digit by 2^n from right to left.
  • Decimal to Octal: Divide by 8 until quotient is 0.
  • Octal to Decimal: Multiply each digit by 8^n from right to left.
  • Decimal to Hexadecimal: Divide by 16 until quotient is 0.
  • Hexadecimal to Decimal: Multiply each digit by 16^n from right to left.

Binary Arithmetic Operations

Binary arithmetic involves addition and subtraction:

  • Binary Addition: Similar to decimal addition but with carry rules for binary.
  • Binary Subtraction: Can be performed using one's complement or two's complement methods.

One’s and Two’s Complement Methods

These methods are used for binary subtraction:

  • One’s Complement: Invert all bits, then add 1.
  • Two’s Complement: Invert all bits, add 1, and discard the carry.

Boolean Algebra

Boolean algebra is a branch of mathematics that deals with logical operations:

  • Boolean Values: True (1) and False (0).
  • Truth Tables: Tables used to evaluate Boolean expressions.
  • Boolean Expressions: Combinations of variables and operators.
  • Boolean Functions: Functions that return Boolean values.

Logic Gates

Logic gates are the building blocks of digital circuits:

  • AND Gate: Returns True if both inputs are True.
  • OR Gate: Returns True if either input is True.
  • NOT Gate: Inverts the input.
  • NAND Gate: Returns True if not both inputs are True.
  • NOR Gate: Returns True if neither input is True.
  • XOR Gate: Returns True if inputs are different.
  • XNOR Gate: Returns True if inputs are the same.

Laws of Boolean Algebra

These laws govern how Boolean expressions can be manipulated:

  • Boolean Identities: A AND A = A, A OR A = A.
  • Complement Laws: A AND NOT A = 0, A OR NOT A = 1.
  • Identity Laws: A AND 1 = A, A OR 0 = A.
  • Commutative Laws: A AND B = B AND A, A OR B = B OR A.
  • Associative Laws: (A AND B) AND C = A AND (B AND C), (A OR B) OR C = A OR (B OR C).
  • Distributive Laws: A AND (B OR C) = (A AND B) OR (A AND C), A OR (B AND C) = (A OR B) AND (A OR C).

Verification of Laws Using Truth Tables

Truth tables are used to verify the validity of Boolean laws by evaluating all possible combinations of inputs.

Applications of Boolean Logic

Boolean logic is fundamental in digital electronics and computer programming, used in designing digital circuits and algorithms. It is also applied in decision-making processes and data analysis.

Conclusion

In conclusion, number systems and Boolean logic are foundational concepts in computer science, enabling the creation of digital systems and algorithms that underpin modern technology.

Boolean logic is used in real-world applications such as:
